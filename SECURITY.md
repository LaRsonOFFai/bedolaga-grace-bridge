# Политика безопасности

Не публикуйте в issue или диагностическом архиве:

- API-токены Remnawave и Bedolaga;
- пароли PostgreSQL;
- Telegram Bot Token;
- содержимое `.env`;
- cookie и JWT;
- персональные данные подписчиков.

Для отчёта используйте `sudo gracectl bundle`: команда удаляет известные
секреты и заменяет UUID пользователей необратимыми отпечатками.

При подозрении на утечку сначала отзовите токены, затем отправьте владельцу
репозитория обезличенный архив. Публичный issue с действующими секретами будет
удалён.
