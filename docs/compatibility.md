# Совместимость Bedolaga

Минимальная версия — **v3.64.0**. Предыдущие версии намеренно не поддерживаются.

Для каждой новой версии требуется запись в
`patches/bedolaga/compatibility.json`, содержащая:

- точный Git commit;
- SHA-256 каждого затрагиваемого исходного файла;
- SHA-256 патча;
- статус `canary` или `stable`.

Версия выше известной не считается совместимой автоматически. Это обязательное
правило: если upstream перенёс оплату или синхронизацию, старый патч может снова
создать гонку.

Пользовательские изменения в файлах, которых патч не касается, попадают в
локальный кандидатный образ. Если изменён затрагиваемый файл, установка
останавливается и предлагает дождаться нового compatibility package.
