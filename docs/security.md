# Модель безопасности

- секреты не принимаются аргументами командной строки;
- `secrets.env`, state, backups и reports создаются с правами `0600/0700`;
- диагностический bundle удаляет токены, пароль DSN, Bearer и UUID;
- patch применяется только к временной копии исходников;
- затрагиваемые файлы проверяются по SHA-256;
- кандидат собирается под новым локальным Docker tag;
- исходный образ остаётся доступен до завершения проверки;
- неизвестная или более новая Bedolaga блокируется;
- `--yes` не отключает compatibility, backup, canary или ступени rollout.
