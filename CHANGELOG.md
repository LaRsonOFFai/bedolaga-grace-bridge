# История изменений

## 0.1.0 — в разработке

- безопасная CLI и мастер конфигурации;
- собственная схема PostgreSQL `grace_bridge`;
- observe, canary и rollout 5/25/50/100;
- дамп PostgreSQL и проверяемый manifest бэкапа;
- безопасный rollback и независимый rescue entrypoint;
- compatibility package для Bedolaga v3.64.0;
- redaction диагностических архивов;
- синтетическая проверка когорты и изолированный PostgreSQL-тест на 40 000 UUID;
- пауза и возобновление с сохранением ступени rollout;
- независимый rescue-модуль на случай повреждения основной CLI;
- автоматический обезличенный отчёт при ошибке установки.
